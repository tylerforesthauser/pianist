# Command Documentation

This directory contains detailed documentation for each Pianist command.

## Available Commands

- [`generate.md`](generate.md) - Generate new compositions from text descriptions
- [`import.md`](import.md) - Import MIDI files to JSON format
- [`modify.md`](modify.md) - Modify existing compositions (transpose, fix, AI modification)
- [`render.md`](render.md) - Render JSON compositions to MIDI files
- [`analyze.md`](analyze.md) - Analyze MIDI or JSON compositions
- [`annotate.md`](annotate.md) - Mark musical intent (key ideas, expansion points)
- [`expand.md`](expand.md) - Expand incomplete compositions to complete works
- [`diff.md`](diff.md) - Compare two compositions
- [`fix.md`](fix.md) - Fix composition issues (pedal patterns, etc.)
