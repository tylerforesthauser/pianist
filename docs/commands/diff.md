## Purpose

Compare two compositions and show what changed.

## Syntax

```bash
pianist diff <input1.json> <input2.json> [options]
```

## Options

- `--format` - Output format: text, json, or markdown (default: text)
- `--musical` - Musical-aware diff (not yet fully implemented)
- `--show-preserved` - Show preserved elements
- `-o, --output` - Output path
- `--verbose, -v` - Show progress
- `--debug` - Show tracebacks
- `--overwrite` - Overwrite existing files
