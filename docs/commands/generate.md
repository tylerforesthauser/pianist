# `generate` Command

**Status**: Placeholder - To be fleshed out

## Purpose

Generate new compositions from text descriptions.

## Syntax

```bash
pianist generate [description] [options]
```

## Options

- `--provider` - AI provider to use (gemini)
- `--model` - Model name (default: gemini-flash-latest)
- `-o, --output` - Output JSON path
- `--render` - Also render to MIDI
- `-m, --midi` - MIDI output path
- `-p, --prompt` - Save prompt template
- `-r, --raw` - Save raw AI response
- `--instructions` - Additional instructions
- `--verbose, -v` - Show progress
- `--debug` - Show tracebacks
- `--overwrite` - Overwrite existing files
