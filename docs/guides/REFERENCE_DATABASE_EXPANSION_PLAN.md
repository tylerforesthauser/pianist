## Sources for Finding MIDI Files

### Classical Works (Public Domain)

1. **IMSLP (International Music Score Library Project)**
   - URL: https://imslp.org
   - Some works have MIDI files
   - All works are public domain
   - Search by composer, work, or catalog number

2. **MuseScore Community**
   - URL: https://musescore.com
   - Filter by "Public Domain" license
   - Many user-uploaded MIDI files
   - Can export as MIDI

3. **Classical MIDI Archives**
   - Various collections available online
   - Search for specific composers/works
   - Examples: Classical MIDI Archives, Kunst der Fuge

4. **Mutopia Project**
   - URL: https://www.mutopiaproject.org
   - Public domain sheet music (some with MIDI)

### Jazz & Ragtime (Public Domain)

1. **Ragtime MIDI Collections**
   - Search for "Scott Joplin MIDI"
   - "Ragtime MIDI archive"
   - Many ragtime works are public domain

2. **Jazz MIDI Archives**
   - Early jazz standards (pre-1924) are public domain
   - Search for "public domain jazz MIDI"
   - Look for piano solo arrangements

3. **Free Music Archive**
   - URL: https://freemusicarchive.org
   - Some jazz and ragtime collections
   - Filter by license type

### Modern & Contemporary

1. **Internet Archive**
   - URL: https://archive.org
   - Search for "piano MIDI" + composer name
   - Many public domain and CC-licensed works
   - Filter by license type

2. **Free Music Archive**
   - Contemporary classical section
   - Filter by CC licenses
   - Some modern works available

3. **MuseScore (CC-Licensed)**
   - Search for contemporary composers
   - Filter by CC-BY or CC-BY-SA license
   - Many users upload CC-licensed arrangements

4. **Composer Websites**
   - Some contemporary composers provide MIDI files
   - Check composer websites for downloadable files
   - Look for CC-licensed works

### Search Strategies

1. **Composer-Specific Searches:**
   - "[Composer name] MIDI"

## Notes

- **Copyright Status:** Always verify copyright status for your jurisdiction. In the US, works published before 1929 are generally public domain. Works by composers who died 70+ years ago may be public domain in some jurisdictions.

- **Quality Over Quantity:** Better to have fewer high-quality examples that clearly demonstrate techniques than many mediocre examples.

- **MIDI File Quality:** Prioritize MIDI files with:
  - Accurate note timing
  - Proper velocity values
  - Correct key signatures
  - Appropriate tempo markings
  - Clean structure (not too many tracks)

- **Technique Clarity:** Choose works where the technique/form is clearly demonstrated, not just present.

- **Diversity Goals:** Aim for representation across:
  - Time periods
  - Geographic regions
  - Musical styles
  - Techniques
  - Forms
