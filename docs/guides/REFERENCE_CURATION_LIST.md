## Sources for MIDI Files

### Recommended Sources

1. **IMSLP (International Music Score Library Project)**
   - URL: https://imslp.org
   - Some works have MIDI files available
   - All works are public domain

2. **MuseScore Community**
   - URL: https://musescore.com
   - Many public domain arrangements
   - Can export as MIDI
   - Filter by "Public Domain" license

3. **Classical MIDI Archives**
   - Various collections available
   - Search for specific composers/works

4. **Free Music Archive**
   - URL: https://freemusicarchive.org
   - Some classical/public domain works

### Tips for Finding Files

- Search for: "[Composer] [Work] MIDI"
- Check MuseScore for user-uploaded arrangements
- IMSLP sometimes has MIDI files linked
- Verify copyright status (composer died 70+ years ago)
- Prefer complete, accurate transcriptions
