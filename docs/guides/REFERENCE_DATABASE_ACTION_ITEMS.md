## Search Tips

### Effective Search Queries

1. **Composer + Work + MIDI:**
   - "César Franck Prélude Choral Fugue MIDI"
   - "Scott Joplin Maple Leaf Rag MIDI"

2. **Composer + Genre + MIDI:**
   - "Gabriel Fauré nocturne MIDI"
   - "Béla Bartók mikrokosmos MIDI"

3. **Work + MIDI:**
   - "Maple Leaf Rag MIDI"
   - "Jeux d'eau MIDI"

4. **Site-Specific Searches:**
   - "site:musescore.com [composer] [work]"
   - "site:imslp.org [composer] [work] MIDI"
   - "site:archive.org [composer] piano MIDI"

### Where to Search

1. **MuseScore:** https://musescore.com
   - Filter by "Public Domain" or "CC" license
   - Many user-uploaded MIDI files

2. **IMSLP:** https://imslp.org
   - Some works have MIDI files
   - All works are public domain

3. **Internet Archive:** https://archive.org
   - Search for "piano MIDI" + composer
   - Filter by license type

4. **Classical MIDI Archives:**
   - Various collections online
   - Search for specific composer/work

5. **Free Music Archive:** https://freemusicarchive.org
   - Filter by genre and license

### Quality Checklist

Before adding a MIDI file, verify:
- [ ] Accurate note timing
- [ ] Proper velocity values
- [ ] Correct key signatures
- [ ] Appropriate tempo markings
- [ ] Clean structure (not too many tracks)
- [ ] Public domain or clearly CC-licensed
- [ ] Clearly demonstrates target technique/form

## Notes

- **Copyright Verification:** Always verify copyright status. In the US, works published before 1929 are generally public domain. Works by composers who died 70+ years ago may be public domain in some jurisdictions.

- **Quality Over Quantity:** Prioritize high-quality MIDI files that clearly demonstrate techniques.
